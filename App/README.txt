To run this application follow the following steps:

1). Open a command shell (must have meteor installed)
2). Navigate to the logbook directory
3). run the command meteor npm install.
4). run the command meteor npm run start.
5). You should be able to connect to the application by opening your browser (we recomend Chrome or Firefox) and typing localhost:3000.
